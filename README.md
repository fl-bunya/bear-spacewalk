# 黄色い熊 Spacewalk

黄色い熊が宇宙飛行士の格好で、発光する地球のそばを宇宙遊泳する
Web アンビエントアート。音楽（既定: 宇宙でさみしいなあ.mp3）をリアルタイム解析し、
音に合わせて世界が脈動する。

```
音 → 映像のマッピング
├─ 低音 (40-260Hz)  → 地球の大気発光・夜の街明かりが脈動 / 照り返しが熊を照らす
├─ 中音 (260Hz-2kHz) → 大気の色相が 青 ⇄ エメラルド に遷移
├─ 高音 (2k-8kHz)    → 星のきらめき
└─ ビート(低音の拍)  → 地球の自転が加速 / 縁から波紋リング / 流星・極オーロラ
```

## 開き方

**推奨**: このフォルダで HTTP サーバを立てて開く（音声解析が確実に動く）

```sh
cd ~/my/earth
npx http-server
# → 表示された http://localhost:8080 を開き、画面をクリック
```

**簡易**: index.html をダブルクリック（file:// 直開き）でも動く。
ただしブラウザによっては file:// で音声解析がゼロを返すことがあり、
その場合は自動で疑似変調（音楽は流れるが解析ではなくLFOで脈動）に
フォールバックする。

※ ブラウザの自動再生制限のため、必ず初回クリックで開始する仕様。

## ファイル構成

```
earth/
├── index.html          本体（シーン・シェーダー・音声解析すべて込み）
├── lib/three.min.js    Three.js r149 UMD版（同梱・ネット接続不要）
├── textures/           地球テクスチャ（Web最適化版: 昼8K/夜4K/雲4K/海面反射2K）
├── materials/          テクスチャ元素材（8K JPEG/TIFF。コミット対象外）
├── music/              BGM置き場（ここの音声ファイルをファイル名順に連続再生）
├── generate-playlist.js  music/ の曲一覧を playlist.json に書き出す（デプロイ用）
├── screenshot.js       ヘッドレス撮影スクリプト（Playwright。動作確認用）
└── README.md
```

## 地球の表示

実写テクスチャ（NASA 画像ベース、Solar System Scope 配布素材 / CC BY 4.0）を
読み込んだ高精細地球を表示する。昼面・夜の街明かり・流れる雲・海面の太陽反射・
大気のフレネル発光を合成しており、夜景と発光は音楽の低音で脈動する。

file:// 直開き等でテクスチャが読み込めない環境では、シェーダー生成の
プロシージャル地球に自動フォールバックする（映像は粗くなるが動作は継続）。

昼夜の地理配置は現在時刻に同期しており（`CONFIG.timeSync`）、
いま現実に夜の地域は画面でも夜になっている。

太陽は実時刻の方向に配置され、地球がその前を巡るたびに日食（縁から光が溢れる
ゴッドレイ）が起きる。カメラが地球の夜側に入ると天の川が浮かび、高緯度の縁には
オーロラがゆらぐ。

カメラは日本上空からスタートし、地球がゆっくり西へ自転して
ユーラシア（中国→中央アジア→ヨーロッパ）を順に映していく俯瞰ビュー。
黄色い熊は常にカメラに同伴し、地平線の少し上を漂う。
自転の速さは曲のビートに連動して加速する。

確認用パラメータ: `?preview`（クリック・音声スキップ）、
`&ry=数値`（初期経度の上書き）、`&t0=秒`（遊覧パスの時刻先送り）。

シャツは既定で無地。`?shirt=文字` を付けると、その文字が
シャツの前面に白抜きで表示される（例: `?shirt=宇宙`）。

キャラクターは `?char=mspn` で mspn（ずんぐり体型・服なし）に
切り替えられる。指定なしは黄色い熊。

## BGM について

`music/` 内の音声ファイル（mp3/m4a/ogg/wav/flac）をファイル名順に連続再生する
（最後まで行ったら先頭に戻る）。曲の追加・削除は `music/` にファイルを
置く／消すだけでよい。

曲一覧は次の優先順で取得する:

1. `music/playlist.json` — 静的ホスティング用。`node generate-playlist.js` で生成
2. HTTPサーバのディレクトリ一覧 — ローカルの http-server / python http.server なら
   playlist.json なしで自動検出される
3. `CONFIG.audioFile` の単曲ループ — file:// 直開き等の最終フォールバック

**画面で差し替え**: 右下の「♪ 曲を選ぶ」ボタン、または画面に mp3 を
ドラッグ＆ドロップすると、その場で BGM を差し替えられる（コード編集不要）。
読み込んだ曲でも音声解析（低音→地球発光など）はそのまま効く。
※ 差し替えはセッション限り（リロードでプレイリストに戻る）。差し替え曲はループ再生。

## 公開（Cloudflare Pages）

https://bear-spacewalk.pages.dev で公開中。デプロイは:

```sh
npm run deploy
```

playlist.json の生成 → `dist/` に公開ファイル（index.html / lib / textures /
music）だけを集める → `wrangler pages deploy` まで一括で行う。
初回のみ `npx wrangler login` で Cloudflare にログインしておくこと。

## 調整ポイント

`index.html` 冒頭の `CONFIG` で主要パラメータを変更できる:
色（bearColor / shirtColor）、地球の大きさ・位置、熊の基準位置、
星の数、自転速度。

3D の黄色い熊は Three.js プリミティブによるコード造形（トゥーン調＋輪郭線）。

## 動作確認用

`index.html?preview` で開くと、クリック演出と音声をスキップして
疑似変調でアニメーションだけ確認できる（ヘッドレス撮影用）。
